# Unit Converter

## Overview

This is a basic conversion app for comparison of metric and imperial units. It is the result of a solo project, "Unit Converter", within the [Fullstack](https://scrimba.com/courses) course at Scrimba. It closely follows a Figma design file that Scrimba links to the project.

## Try it out

> Clone the repository to your local machine.

> Install the dependencies and run the project:

```
npm install
npm start
```

Visit https://vitejs.dev/ for useful information on Vite
